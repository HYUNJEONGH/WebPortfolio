역할: 안드로이드 개발, 서버 구축

한이음 공모대전 2016에 참여한 비콘 기반의 간편결제 시스템이라는 프로젝트에서 안드로이드 개발과 서버 구축을 맡아 진행했습니다. 비콘을 활용한 간편결제 기술을 시스템에 적용하는 것이 목표였습니다.
서버는 리눅스 환경에 DB는 MySQL를 사용하여 구축 했고 PHP로 통신을 하였습니다. 서버에서 JSON Data를 전송받아 어플을 사용하는 사용자에게 관련 내용을 출력, 결제 기능 (결제 완료 신호만 받음) 적용, 결제된 내용을 서버에 보내어 저장, Beacon과 통신을 통해 팝업 창을 띄워주고 LBS 기술을 적용시켜 좌석을 안내해주는 기능을 가지고 있습니다.
